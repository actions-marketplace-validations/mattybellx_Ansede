# Ansede Static Documentation

Ansede Static is an offline-first SAST engine focused on high-signal detection of authorization and taint-flow vulnerabilities across modern codebases.

## What you can do here

- Install and run Ansede quickly on local code or CI
- Explore the rule database and severity model
- Understand taint engine internals and architecture
- Author custom YAML rules safely with schema validation
- Track benchmark quality and roadmap progress
